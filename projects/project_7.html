Sealer Deck Quality Improvements

The sealer deck is a partially automated assembly line where operators apply sealer onto different parts of the chassis in order to help waterproof the interior of a car. One common problem seen at this location is the use of sealer, and the quality of applications. The purpose of this project was to reduce the effects of these issues.

My primary role in this project was to design and implement tools that can be used on the sealer deck to make the operator's lives easier, while improving part quality. I did this by creating the tools below.

The first thing I made was a paint brush cleaner that operators could quickly wipe their brushes with. This prevented the use of rags, decreased cycle times, and collected excess sealer to be used again!

The next thing I made was a re-usable hole plug that prevented excess sealer from entering areas that caused assembly issues down the assembly line.

I also created handheld sealer cups and sealer cup attachments operators can use to scrape their tools clean with, as well as place any extra tools in the provided holders.
